Basic Usage

Use the o-grid class on the wrapping parent. Then use o-col-[columnSize] on internal elements to take up different column widths:

Wrapping columns

Columns will wrap

Responsive columns

Column sizes can change based on class name. For each breakpoint name and size added to the $column_breakpoints map variable, a column class with that name and width will be avaiable.

Example:

$column_breakpoints: (
  sm: 600px,
  md: 800px,
  lg: 1000px
);

This map allows for column classes o-col-sm-[columnSize], o-col-md-[columnSize], o-col-ld-[columnSize]

Offsetting columns

Column off sets can be controlled via the o-col-start-[columnStart] class.

Similar to responsive column classes, offset classes can also be used based according to the $column_breakpoints map variable, with the breakpoint size being before the columnStart value: o-col-start-[breakpoint]-[columnStart]

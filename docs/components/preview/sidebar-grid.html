Sidebar Grid

To use the sidebar grid, configure vairables in css, then use o-grid-sidebar as the primary wrapper class, o-grid-sidebar__main for the primary content wrapper, and o-grid-sidebar__aside for the sidebar content wrapper.

The o-grid class can then be used inside of the primary and aside containers as needed.

You may wish to further target aspects of the o-grid class when inside the primary/aside containers (change the number of grid columns, etc), but that can easily be done per project or per component. The same is true for the o-col-* classes.

Edge to Edge Backgrounds

Basic Grids within wrappers can have background colors that run edge to edge, just make sure that the $grid-sidebar-use-gap variable is set to false.

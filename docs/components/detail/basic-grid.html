<div class="frctl-doc-block o-grid">
    <div class="frctl-doc-block__content o-col-10">
        <h2>Responsive columns</h2>
        <p>Column sizes can change based on class name. For each breakpoint name and size added to the <code>$column_breakpoints</code> map variable, a column class with that name and width will be avaiable.</p>
        <p>Example:</p>
        <pre><code>$column_breakpoints: (
  sm: 600px,
  md: 800px,
  lg: 1000px
);</code></pre>
        <p>This map allows for column classes <code>o-col-sm-[columnSize]</code>, <code>o-col-md-[columnSize]</code>, <code>o-col-ld-[columnSize]</code>
        <p>
    </div>
</div>

        <p>Column off sets can be controlled via the <code>o-col-start-[columnStart]</code> class.</p>

<div class="frctl-doc-block o-grid">
    <div class="frctl-doc-block__content o-col-10">
        <h2>No Gap Grid</h2>
        <p>Add the <code>o-grid--no-gap</code> class to set the native grid <code>gap</code> selector to zero.</p>
        <p>This class removes the gap from the grid, and adds padding to the columns. This allows the columns to have edge to edge color, but keep content inline with the grid.</p>
    </div>
</div>

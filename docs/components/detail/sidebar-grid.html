    <div class="frctl-doc-block__content o-col-10">
        <h2>Sidebar Grid</h2>
        <p>To use the sidebar grid, configure vairables in css, then use <code>o-grid-sidebar</code> as the primary wrapper class, <code>o-grid-sidebar__main</code> for the primary content wrapper, and <code>o-grid-sidebar__aside</code> for the sidebar content wrapper.</p>
        <p>The <code>o-grid</code> class can then be used inside of the primary and aside containers as needed.</p>
        <p>You may wish to further target aspects of the <code>o-grid</code> class when inside the primary/aside containers (change the number of grid columns, etc), but that can easily be done per project or per component. The same is true for the <code>o-col-*</code> classes.</p>
    </div>
</div>
